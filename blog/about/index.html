About this blog

Posted on Apr 15, 2025

This blog is my first foray into using the static web site generator Hugo, in combination with a custom domain from Squarespace and some help from Github Pages. There are a zillion different ways to write blogs these days and even more ways to host/serve them, but for better or worse this is what I’ve picked out. For now :-).

I’ve intentionally set up a robots.txt file to disable search engine indexing, so in theory it should not get crawled by Google or any other search engine (this is mainly in the interests of privacy, since I’ll be posting photos of myself and other riders at various points).
